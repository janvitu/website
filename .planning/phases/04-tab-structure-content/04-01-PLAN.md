Create a new file `hugo/layouts/shortcodes/tab.html` -- a paired Hugo shortcode template.

The shortcode accepts three named parameters: `id`, `label`, and optional `active`.

Output structure:
```html
<div
  role="tabpanel"
  id="panel-{{ .Get "id" }}"
  aria-labelledby="tab-{{ .Get "id" }}"
  class="tab-panel"
  data-tab-id="{{ .Get "id" }}"
  {{- if not (.Get "active") }} hidden{{ end }}
>
  {{ .Inner | safeHTML }}
</div>
```

Key details:
- Use `{{ .Inner | safeHTML }}` (not bare `{{ .Inner }}`) to prevent HTML entity double-escaping (per RESEARCH.md Pitfall 2)
- When `active="true"` is passed, omit the `hidden` attribute so the panel is visible on page load (per RESEARCH.md Pitfall 3)
- When `active` is not passed or is falsy, include `hidden` attribute
- Use backtick parameter syntax: `.Get "id"` (matches existing linkStrike.html pattern)
- Add a Hugo template comment at the top: `{{/* Tab panel shortcode - paired. Params: id, label, active (optional) */}}`
- Use tabs for indentation, not spaces (per Prettier config)
